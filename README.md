# Escape-Animal-Clasifica-o-Muere-
Escape Animal: Clasifica o Muere es un videojuego donde el jugador, en un mundo postapocalíptico, debe clasificar animales en tres corrales según sus características. Basado en teoría de conjuntos, el jugador gana puntos o pierde vidas según sus decisiones. Proyecto grupal del curso de matemáticas discretas en Jala University.
